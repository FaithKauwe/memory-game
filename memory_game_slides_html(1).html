⭐ Socket.io Events Architecture

Client → Server Events

🐰
player_joined
Player connects to game
🦊
start_game
Game initialization with difficulty
🐻
flip_card
Player attempts to flip a card
🐼
end_turn
Player passes turn to opponent
🐨
reset_game
Reset game state

Server → Client Events

🐯
player_joined
Broadcast new player
🦁
player_left
Broadcast player disconnect
🐮
game_start
Send game board to all players
🐷
card_flipped
Broadcast card flip to all
🐸
card_flipped_face-down
Broadcast card flip back
🐵
match_found
Celebrate matching pair
🐔
no_match
Notify no match found
🐧
turn_change
Switch active player
🐦
invalid_turn
Reject out-of-turn action
🦉
turn_locked
Prevent extra flips
🦆
game_over
End game with winner
🦅
game_reset
Clear board
